## 프로그램 개요

### 🎯 핵심 목표

**"경험을 통한 학습, 시뮬레이션을 통한 성장"**

단순히 이론을 배우는 것이 아니라, 가상 환경에서 실제와 유사한 상황을 체험하고 AI와 상호작용하며 문제 해결 능력을 키우는 몰입형 교육 프로그램

### 📊 프로그램 특징

| 구분 | 내용 |
|------|------|
| **대상** | 실전 경험을 원하는 AI 학습자, 의사결정 능력 향상 희망자 |
| **기간** | 각 소주제당 8-12주 (총 24-36주) |
| **방식** | 가상환경 몰입 + 시뮬레이션 + AI 피드백 |
| **결과물** | 실전 대응 능력 + 포트폴리오 + 최종 분석 보고서 |

### 🌟 차별화 요소

```mermaid
graph LR
    A[기존 교육] --> A1[이론 중심]
    A --> A2[단편적 실습]
    A --> A3[제한된 피드백]
    
    B[가상환경 교육] --> B1[체험 중심]
    B --> B2[연속적 시나리오]
    B --> B3[실시간 AI 반응]
    
```

```mermaid
graph TB
    subgraph "사용자 레이어"
        U[학습자] --> UI[웹/앱 인터페이스]
    end
    
    subgraph "AI-LLM 레이어"
        ENV[가상환경 구축 AI]
        REACT[반응 생성 AI]
        REPORT[보고서 작성 AI]
    end
    
    subgraph "데이터 레이어"
        JSON[환경 시나리오 JSON]
        USER_DATA[사용자 행동 데이터]
        ANALYTICS[분석 결과 DB]
    end
    
    UI --> ENV
    ENV --> JSON
    
    UI --> REACT
    REACT --> USER_DATA
    JSON --> REACT
    
    USER_DATA --> REPORT
    ANALYTICS --> REPORT
    REPORT --> UI
    
    style ENV fill:#4ECDC4
    style REACT fill:#FF6B6B
    style REPORT fill:#95E1D3
```

### 📦 시스템 구성 요소

| 구성 요소 | 역할 | 핵심 기술 |
|----------|------|----------|
| **가상환경 구축 AI** | 안정적인 시나리오 환경 생성 | 사전 정의 JSON + LLM 검증 |
| **반응 생성 AI** | 현실적이고 일관된 NPC/상황 반응 | GPT-4 + 컨텍스트 관리 |
| **보고서 작성 AI** | 행동 패턴 분석 및 인사이트 도출 | 데이터 분석 + 자연어 생성 |

## 핵심 페르소나

### 👤 페르소나 1: 민준 (29세, 스타트업 대표)

**프로필:**
- 직업: 에듀테크 스타트업 공동창업자
- 특징: 기술은 좋지만 협상/비즈니스 경험 부족
- 고민: "투자 유치 미팅이 두렵다", "팀 관리가 어렵다"

```mermaid
mindmap
  root((민준))
    현재 상태
      기술 역량 우수
      비즈니스 경험 부족
      의사결정 불안
    Pain Points
      투자자 협상 미숙
      팀 갈등 해결 어려움
      전략적 판단 부족
    학습 목표
      협상 능력 향상
      리더십 개발
      위기 대응력
    활용 소주제
      비즈니스 협상
      위기 상황 대응
```

**유저 시나리오 - 투자 유치 준비 과정:**

```
Day 1: 가상환경 시작
- 민준이 플랫폼에 로그인
- "시리즈 A 투자 유치" 시나리오 선택
- 가상환경 AI가 VC 투자자 3명 생성 (보수형, 공격형, 중립형)
- 초기 상황: "회사 가치 평가 50억, 투자 희망 금액 10억"

Day 1-2: 첫 번째 미팅
- 보수적 투자자와 미팅 시작
- 민준: "저희 서비스는..."
- AI 반응: "시장 규모가 작아 보이는데요" (표정: 회의적)
- 민준이 답변 선택 또는 직접 입력
- AI가 실시간으로 투자자 반응 생성
- 결과: 추가 미팅 약속 (진행도 30%)

Day 3-4: 재협상 준비
- 중간 피드백 리포트 수신
  "당신은 기술에만 집중했습니다"
  "시장 데이터 제시가 부족했습니다"
  "제안: 경쟁 분석 자료 준비"
- 개선된 자료로 두 번째 미팅

Day 5-7: 최종 협상
- 여러 시도 끝에 조건 협의
- 지분 희석률, 이사회 구성 등 논의
- 최종 결과: 성공/실패 + 상세 분석

최종 보고서 수신:
- "7번의 시도 끝에 성공"
- "당신의 강점: 빠른 학습, 데이터 기반 설득"
- "개선점: 감정 관리, 대안 제시 능력"
- "실제 투자 미팅 전 체크리스트"
```

**유저 시나리오 - 글로벌 프로젝트 착수:**

```
Week 1: 킥오프 미팅
- 시나리오: "미국-한국-독일 팀과 신규 프로젝트"
- 가상 환경: 각국 팀원 3-4명 (AI 캐릭터)
- 첫 화상회의 시뮬레이션

미팅 중:
- 미국 팀원(Jake): "Let's move fast and break things"
- 한국 팀원(수진): "계획을 먼저 세밀하게..."
- 독일 팀원(Klaus): "프로세스 문서가 필요합니다"
- 지혜가 중재 및 방향 제시 필요

AI 반응 시스템:
- 지혜의 발언에 따라 각 캐릭터 반응 변화
- Jake: 빠른 결정 선호 → 지연시 불만 표시
- 수진: 안정성 중시 → 급진적 변화에 저항
- Klaus: 규칙 준수 → 무계획 진행시 협조 거부

Week 2-3: 갈등 발생
- 일정 지연 이슈
- 문화적 오해로 인한 긴장
- 각 팀원의 불만 AI가 현실적으로 표현
- 지혜의 해결 시도

Week 4: 해결과 성장
- 여러 시도를 통한 조율
- 문화 차이 이해 및 극복
- 프로젝트 진행 정상화

최종 보고서:
- "초기 3번의 미팅에서 문화 충돌 발생"
- "4번째 시도에서 각 팀의 강점 활용 전략 성공"
- "학습한 스킬: 경청, 맥락 파악, 유연한 의사결정"
- "실제 글로벌 프로젝트 가이드라인"
```
