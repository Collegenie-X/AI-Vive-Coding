## 🎯 프로그램 핵심 철학

### "먼저 만들고, 실패하면서 배우고, 비즈니스로 완성한다"

```mermaid
graph LR
    A[아이디어] --> B[즉시 구현]
    B --> C[실패/문제 발견]
    C --> D[이론 탐색]
    D --> E[개선]
    E --> F[비즈니스 모델]
    F --> G{시장 검증}
    G -->|No| C
    G -->|Yes| H[런칭]
    
    style A fill:#ffe66d,color:#f00
    style C fill:#ff6b6b
    style F fill:#4ecdc4,color:#111
    style H fill:#95e1d3,color:#111
```

**역진행 학습 방정식**:
```
빠른 시제품 제작 × 실패 기반 학습 × 비즈니스 사고 = 기업가형 개발자
```

### 왜 "만들기 먼저" 방식인가?

전통적 교육은 **이론 → 실습 → 프로젝트**지만, AI 시대는 다릅니다.

- ❌ 기존: 6개월 배우고 → 1개월 만들기
- ✅ 새로운: 1일 만들고 → 6개월 개선하며 배우기

**미네르바 스쿨 방식**: 능동적 참여, 실전 문제 해결, 다각도 사고
