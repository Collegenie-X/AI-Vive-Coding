# 바이브 코딩 기반 기업가정신 교육 프로그램

## 프로그램 개요

### 목적
바이브 코딩(자연어 기반 AI 코딩)을 활용하여 학생들이 **기업가정신**과 **실행력**을 갖춘 창작자로 성장하도록 지원합니다.

### 핵심 철학
- **기획 중심 학습**: 코드 작성보다 문제 정의, 사용자 이해, 솔루션 설계에 집중
- **빠른 프로토타이핑**: AI 도구로 아이디어를 즉시 구현하며 실행력 강화
- **실전 경험**: 실제 사용자 피드백을 받고 개선하는 전 과정 체험
- **자유로운 창작**: PBL(프로젝트 기반 학습) 방식으로 학생 주도 프로젝트 진행

### 사용 도구
- **Google AI Studio**: 아이디어 구체화, 코드 생성, 문제 해결 지원
- **V0 (Vercel)**: 자연어로 UI/UX 프로토타입 빠르게 생성
- **Cursor**: AI 페어 프로그래밍으로 프로젝트 완성 및 디버깅

### 전체 구조
- **9학년**: 자기표현과 창의적 문제 발견 (8차시)
- **10학년**: 사회 공감과 협력적 문제 해결 (8차시)
- **11학년**: 창업 실행과 혁신적 솔루션 설계 (8차시)
- **총 24차시**: 각 학년별 완성형 프로젝트 1개 산출

## 9학년: 자기표현과 창의적 문제 발견 (8차시)

### 학습 목표
1. 자신의 관심사와 강점을 탐색하고 디지털 콘텐츠로 표현할 수 있다
2. 바이브 코딩 도구를 활용하여 아이디어를 웹 프로토타입으로 구현할 수 있다
3. 사용자 관점에서 피드백을 수용하고 개선할 수 있다
4. 창의적 문제 발견 능력과 자기주도적 학습 태도를 기른다

### 프로젝트 주제
**"나를 표현하는 인터랙티브 웹 포트폴리오"**
- 자신의 이야기, 관심사, 꿈을 담은 개인 웹사이트 제작
- 인터랙티브 요소(퀴즈, 게임, 타임라인 등) 포함

## 10학년: 사회 공감과 협력적 문제 해결 (8차시)

1. 사회 문제를 발견하고 공감 능력을 바탕으로 해결책을 제안할 수 있다

**"우리 지역/학교 문제 해결 웹 플랫폼"**

### 차시별 진행 계획

#### 1차시: 팀 빌딩 및 문제 발견
**목표**: 팀을 구성하고 해결할 문제 찾기

**활동 내용**

   - 각 문제의 심각성과 해결 가능성 평가
   - 상위 3개 문제 선정

3. 문제 정의 워크시트 작성 (15분)

   - 이 문제가 해결되면 어떤 변화가 생기는가?
   - 우리가 해결할 수 있는 이유는?

| **미흡** (3-4점) | 도구 활용에 어려움, 기본 기능만 구현 |
| **매우 미흡** (1-2점) | 도구 사용 미숙, 동작하지 않는 부분 많음 |

- 정기적 팀 회고 시간 마련

**해결책**

- 에러 메시지를 AI에게 보여주고 해결책 요청

## 맺음말

이 프로그램은 단순히 코딩을 가르치는 것이 아니라, **문제를 발견하고 해결하는 실행자**를 키우는 것을 목표로 합니다.

바이브 코딩이라는 혁신적 도구를 통해 학생들은:
- 기술적 장벽 없이 아이디어를 빠르게 구현하고
- 실제 사용자 피드백을 받으며 학습하고
- 실패와 개선을 반복하며 성장합니다

**3년 후, 학생들은:**
- 자신의 아이디어를 스스로 실현할 수 있는 실행력
- 사용자 중심으로 생각하는 공감 능력
- 데이터 기반으로 의사결정하는 논리적 사고
- 팀과 협력하여 더 큰 가치를 만드는 협업 능력
- 실패를 두려워하지 않는 도전 정신

을 갖춘 **미래의 혁신가**로 성장할 것입니다.
